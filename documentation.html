Introduction

AquaWind is a modern, responsive HTML5 template designed specifically for kitesurfing businesses, schools, and events. Built with Tailwind CSS, it offers a clean, professional design with beach-inspired colors and smooth animations.

This template includes 9 essential pages, is fully responsive, and follows modern web standards for optimal performance and SEO.

Setup Instructions

1. Basic Setup

  • Upload all files to your web server
  • Ensure all HTML files are in the root directory
  • Create an 'assets' folder for images and other media
  • Test all navigation links work correctly

2. Tailwind CSS Setup

The template uses Tailwind CSS via CDN for easy setup. For production:

  • Install Tailwind CSS locally for better performance
  • Use the included tailwind.config.js file
  • Build your CSS file for production

3. Images & Media

  • Add your kitesurfing images to assets/img/
  • Update image placeholders throughout the template
  • Optimize images for web (recommended: WebP format)
  • Ensure alt text is added for accessibility

Customization Guide

Colors

The template uses a beach-inspired color palette:

ocean-blue: #0077be
aqua: #00bcd4
sand: #f5f1eb
sunset: #ff6b35
wave: #4dd0e1

To change colors, update the tailwind.config.js file.

Typography

  • Primary Font: Inter (body text, navigation)
  • Display Font: Playfair Display (headings, titles)
  • Both fonts are loaded from Google Fonts
  • To change fonts, update the Google Fonts link and Tailwind config

Logo & Branding

  • Replace "AquaWind" with your business name in all HTML files
  • Update the logo SVG or replace with your image logo
  • Ensure the logo is consistent across all pages
  • Update the favicon.ico file

Content Sections

Each section is clearly commented in the HTML:

  • Hero sections with background images
  • Service/feature cards
  • Testimonials and reviews
  • Contact forms with validation
  • FAQ sections with accordion functionality
